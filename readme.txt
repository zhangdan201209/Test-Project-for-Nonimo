//This project can be runing both with android 2.2 and android 4.4


1 . go to profileServere folder
    type in "npm start" for start the server

2.  go to  /ForNonimo/res/values/string.xml
    change the <string name="urlServer">http://192.168.0.111:3000/profile</string>
    with the current ip address of your own server ip address.
    Then recomplie the whole android project.
